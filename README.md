# hey-you

A macOS companion that keeps you honest.

When you start a session, tell it what you're working on. It lives as a tiny dot
on your screen, watches what you're doing, and gently calls you out when you
drift into doomscroll territory — using your Mac's voice.

```
🟢 Focused   →   🟡 Hmm...   →   🔴 Caught you
```

Built with Swift, AppKit, and a conversational LLM over OpenRouter.
